# AI Learning Track

Azure-centric AI engineering roadmap (target: 5+ yrs senior/lead AI role).
Each session: analogy → concepts → local hands-on project → industry standards →
interview Qs. Projects run offline by default with Azure wired behind env vars.

## Required for every project
- `README.md` (run + concepts table)
- `docs/INDUSTRY_STANDARDS.md` — **mandatory.** Maps each industry-standard
  practice to exact code (file + line), explains why it helps, gives the
  interview angle. Linked from the README.
- `docs/AZURE_INTEGRATION.md` — placeholders for real Azure resources
- `tests/` — pytest, runnable offline
